Windows / macOS / Linux

Go to https://www.python.org

Download the latest Python 3 version

Install it

✅ Windows users: Check “Add Python to PATH” before clicking Install

Verify Installation

Open a terminal or command prompt and run:

python --version


or

python3 --version

2. Install Visual Studio Code (VS Code)

Go to https://code.visualstudio.com

Download and install VS Code

Open VS Code

3. Open Project in VS Code

Open VS Code

Click File → Open Folder

Select the folder that contains your Python file

Open the folder

(Optional but recommended)

Install the Python extension by Microsoft from the Extensions tab

4. Run Python File
Using Terminal in VS Code

Open the terminal in VS Code:

View → Terminal

Run the command:

python file-name.py


If that doesn’t work, try:

python3 file-name.py

Example
python main.py

Done ✅

Your Python program should now run successfully.
